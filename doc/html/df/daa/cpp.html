C++ Part

Description of the C++-Part

The C++-Part contains of two different parts, a part for reading the cpu-information and a part to create a model out of this information.

Reading the cpu information

The classes for reading the cpu are cpuInfoBase and linuxCpuInfo. The abstract base class can be derived to add a new cpuInfo for another system, for example windows.

Datamodel of the cpu information

The dataModel is the interface between the cpu information classes and the qml part. The classes which provide the model are qmlCpuInfoItem, cpuInfoModel and qmlCpuInfoModelContainer
